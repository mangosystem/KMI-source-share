  - 지역 해양환경 및 산업 관련 이슈 대시민 서비스 제공

## 구동환경
  - node v18 or later


## 프로젝트 실행 방법
  1. 프로젝트 클론
  1. npm install
  1. npm start


## 배포 방법
  1. npm run build
  1. dist 폴더 내 생성파일 전부 배포 위치로 복사
    * 배포 위치: /home/project/smart_village/webapps/smart_village/
